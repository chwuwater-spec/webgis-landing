# WebGIS 停水方案實戰課｜Landing Page

自來水事業 WebGIS 停水方案課程的銷售頁。

## 檔案

- `index.html` — 單檔 HTML（Tailwind CDN + 內嵌 CSS/JS，直接部署即可）
- `answers.json` — 生成頁面時使用的答案快照

## 部署

靜態站點，可直接部署到 Cloudflare Pages / GitHub Pages / Vercel 等任何靜態託管服務。

### Cloudflare Pages 設定
- Framework preset: **None**
- Build command: （留空）

## 修改文案
